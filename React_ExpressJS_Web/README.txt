This app is made using Express using node.js and react. 

Run these commands before you start this app.
Reach this path: React_ExpressJS_Web/api
run this command
-npm install
Reach this path: React_ExpressJS_Web/client
run this command
-npm install

For the backend (API): 
reach the api folder path in a command terminal and use: -npm start

For the frontend (React):
reach the client folder path in a command terminal and use: -npm start
